 ## Sejarah python dan Apa itu Python?

 Python mulai dikembangkan oleh [ Guido van Rossum ](https://id.wikipedia.org/wiki/Guido_van_Rossum)Pada tahun 1990 di Stichting Mathematisch Centrum (CWI) dan versi terakhir hasil pengembangannya sampai pada versi 1.2 sampai saat ini python sudah berkembang sampai dengan versi 3.6.3 dan akan terus dikembangkan. Lalu apa sebenarnya python itu?. **Python merupakan** bahasa pemrograman yang multiguna karena dengan python semua jenis program bisa kita buat mulai dari website, game, aplikasi mobile, dsb. ## Persiapan Belajar Pyhton di Android

 Untuk alat dan bahannya sebagai berikut: 1. **[Termux](https://www.kumpul4semut.com/termux):** Aplikasi ini sekarang memang sedang hangat-hangatnya dibicarakan oleh para programer karena kehandalannya untuk menjalankan berabagai macam bahasa pemrograman bukan hanya python.
2. **Text Editor:** Banyak sekali aplikasi untuk edit text di android. Disini kita akan menggunkan aplikasi QuickEdit untuk membuat kode pythonnya nanti.
 
## Bagaimana Install Python di Termux?

 Untuk menjalankan python di termux tentunya kita harus menginstall dahulu packet nya di termux. dengan cara seabgai berikut: 
 ```bash
pkg install python
```

 Untuk mengecek apakah pythonnya sudah terinstall belum kalian jalankan perintah **python -v** di termux. 
```bash
python -v
python 2.7.12
```

## Kenapa Python?

 Kenapa kita sebagai pemula disarankan untuk mempelajari bahasa python kenapa bukan bahasa lain seperti php? 
1. Karena python mudah dipelajari bahkan hanya dengan android.
2. Karena penulisan kodenya itu sangan simpel dan cenderung pendek.
3. Bahasa pemrograman multiguna.
4. Kodenya rapi dan tidak berbelit-beli.
 
 Itu beberapa alasan kenapa kita harus belajar python. Gimana lanjut gak mau belajar python. Karena python sudah versi 3 tapi disini kita akan belajar python dasar atau python1 agar kita bisa membandingankan perubahannya gimana di python3.
