  ### Apa Itu Git dan Github?

  Git merupakan sebuah Version Control System ( VCS ) yang biasanya digunakan oleh tim pengembang perangkat lunak. Git akan mencatat semua perubahan yang terjadi pada source code yang dibuat. Tidak hanya itu git juga memudahkan kita dalam bekerja tim.

  Sedangkan github merupakan website VCS yang dasarnya menggunakan git. Jadi antara git dan github ini biasanya sangat sering dipakai oleh programer dalam membuat sebuah aplikasi. Yang membedakannya cuma kalo git itu bersifat offline sedangkan github bersifat online. Tapi keduanya bisa kita kombinasikan untuk keperluan kita dalam menulis source code program.

  ### Alat dan Bahan Untuk Belajar Git dan Github di Android

  Zaman sekarang pasti semua orang punya smartphone. Alangkah baiknya kita gunakan smartphone untuk hal yang lebih bermanfaat. Untuk alat dan bahannya sebagai berikut:

  - **Aplikasi termux** - Siapa yang tak kenal dengan aplikasi yang hebat ini. Aplikasi termux sudah mulai terkenal sejak rilisnya di play store bagi yang belum instal aplikasi termux silahkan download dulu <https://play.google.com/store/apps/details?id=com.termux>
- **Akun Github -** Karena kita akan langsung sekaligus belajar git nya dengan github maka kalian harus membuat akun githubnya juga bagi yang belum punya akunnya silahkan buat di <https://github.com/join?source=header-home>
 
  ### Cara Install Git di Termux

  Sebelum kita menjalankan fungsi-fungsi git kita harus install terlebih dahulu git nya di termux kita. Untuk menginstallnya kalian harus siapin akses internet lalu masukan command!

   ```
<pre class="lang:default decode:true ">$ pkg install git
```

  Lalu untuk mengecek apakah git sudah terinstall di termux masukan command!

   ```
<pre class="lang:default decode:true ">$git --version
```

Jika muncul seperti gambar dibawah ini makan git sudah berhasil di install di termux kita
