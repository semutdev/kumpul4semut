## Apa Itu Termux?

 Termux merupakan terminal emulator untuk android dan merupakan aplikasi lingkungan linux yang bekerja secara langsung tanpa perlu rooting atau setup. Yang membedakan termux dengan aplikasi terminal android lainnya yaitu paket dalam termux itu lebih luas dan banyak. ## Apakah Bisa Termux Digunakan Untuk Hacking?

 Ya termux bisa melakukannya karena termux mempunyai banyak alat yang dapat digunakan untuk meretas. Seperti Hydra ( Brute Force Tool ) , nmap ( Pemindai Jaringan ) dsb. Namun kami menyarankan untuk digunakan untuk hal yang baik saja kalian bisa melakukan peretasan kepada aplikasi yang telah kalian buat dan kalian ingin melakukan pengecekan keamanan untuk aplikasi anda. ## Fungsi dan Manfaat Termux

 Ada berbagai macam terminal emulator yang sudah dibuat untuk dipakai di android. Namun termux ini beda dari yang lain karena banyak sekali paket yang bisa dijalankan di termux. Dan banyak bahasa pemrograman pun bisa dijalankan dengan lancar di aplikasi termux ini. Berikut beberapa fungsi termux: 1. Bisa menjalankan bash shell script layaknya di linux dan zsh shell.
2. Bisa mengakses server melalui ssh. Jadi buat yang mempunya vps kalian bisa juga meremote vps kalian menggunakan ssh melalui termux ini.
3. Bisa menjalankan game berbasis teks dengan frotz.
4. Bisa menjalankan berbagai macam bahasa pemrograman mulai dari c, Python, javascript, java dsb.
5. Mendukung edit file vim dan nano.
6. dsb.
 
 Adapun manfaat dari termux sebagai berikut: - Untuk mengontrol website karena sudah mendukung git dsb.
- Mampu menjalankan tools yang bisa dipakai di linux misalnya tools untuk penestrasi metasploit, Nmap, dsb.
- Untuk mengamati proyek yang sedang dibuat di github.
- Untuk belajar progarming.
 
## Cara Menggunakan Termux Pemula

 Buat anda yang belum pernah mencoba termux apalagi anda adalah seorang programer ada baiknya anda mencoba aplikasi yang satu ini. Tidak hanya programer siapa saja bisa mempelajarinya karena pasti sekarang semua sudah mempunyai smartphone. Berikut langkahnya bagi pemula untuk mencoba aplikasi keren satu ini: ### Install Aplikasi Termux

 Untuk mendapatkan aplikasi ini kalian tinggal install di play store cek link nya [Download Termux](https://play.google.com/store/apps/details?id=com.termux) .Namun Kekurangan aplikasi termux ini hanya mampu berjalan di android versi 5.0 atau android lollypop. Untuk versi android dibawahnya belum bisa menggunakan aplikasi termux. ### Mengizinkan Akses Penyimpanan Untuk termux

 Setelah kalian install aplikasi termux. Ada satu pengaturan yang harus kalian atur terlebih dahulu karena termux belum secara otomatis mengaturnya. Hal yang perlu di atur adalah izin penyimpanannya. Jika kalian tidak mengizinkan penyimpanannya maka kalian tidak bisa mengakses penyimpanan kalian melalui termux ini. Cara mengizinkan akses penyimpanan termux: 1.Buka pengaturan handphone lalu cari aplikasi termux di menu aplikasi yang ada di pengaturan handphone. [![](https://www.kumpul4semut.com/wp-content/uploads/2019/07/20190703_141523-1024x576.png)](https://www.kumpul4semut.com/wp-content/uploads/2019/07/20190703_141523.png) 2.Pilih aplikasi termux tersebut sampai muncul info aplikasi lalu cari menu izin penyimpanan seperti pada gambar dibwawah ini. [![](https://www.kumpul4semut.com/wp-content/uploads/2019/07/20190703_141649-1024x576.png)](https://www.kumpul4semut.com/wp-content/uploads/2019/07/20190703_141649.png) 3.Pilih menu izin tersebut lalu izinkan penyimpanan untuk diakses termux. [![](https://www.kumpul4semut.com/wp-content/uploads/2019/07/20190703_141724-1024x576.png)](https://www.kumpul4semut.com/wp-content/uploads/2019/07/20190703_141724.png) Oke untuk mecoba mengakses penyimpanan internal di termux kalian bisa masukan command! ```

```

 lalu kalian masukan! ```

```
